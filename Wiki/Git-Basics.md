# Git Basics

## Initilization (Only Once)
1) Create New Folder
 `mkdir RBE501`
2) `cd RBE501` 
3) `git init`
4) `git remote add mtm https://github.com/sapan-ostic/RBE-_501_Project.git`
5) `git pull mtm master`

## Commiting changes
1) `git pull mtm master`
2) `git add -A` 
3) `git commit -m "write commit message here"`
4) `git push mtm master`
